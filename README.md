# blink-cmp-zsh

ZSH completion source for blink.cmp

‼️ This project is still in a very early development stage. Please don't use this in prod :D. ‼️

## Local Development

```
vim.opt.runtimepath:prepend("~/PATH_TO_REPO/bink-cmp-zsh")
```
